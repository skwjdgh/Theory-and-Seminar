## 2. 리스트 (List)

### 2.1 선형 리스트의 특징

- **순서**: 요소들이 순서를 가짐
- **동적 크기**: 실행 중에 크기 변경 가능
- **임의 접근**: 인덱스를 통한 직접 접근

### 2.2 리스트 ADT

```python
class List:
    def __init__(self):
        self.items = []
    
    def insert(self, pos, item):
        """지정된 위치에 요소 삽입"""
        self.items.insert(pos, item)
    
    def delete(self, pos):
        """지정된 위치의 요소 삭제"""
        if 0 <= pos < len(self.items):
            return self.items.pop(pos)
        raise IndexError("Invalid position")
    
    def search(self, item):
        """요소 검색"""
        try:
            return self.items.index(item)
        except ValueError:
            return -1
    
    def get_entry(self, pos):
        """지정된 위치의 요소 반환"""
        if 0 <= pos < len(self.items):
            return self.items[pos]
        raise IndexError("Invalid position")
    
    def is_empty(self):
        """리스트가 비어있는지 확인"""
        return len(self.items) == 0
    
    def size(self):
        """리스트 크기 반환"""
        return len(self.items)
    
    def clear(self):
        """리스트 초기화"""
        self.items = []
```


### 2.3 연결 리스트 (Linked List)

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None
        self.size = 0
    
    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = new_node
        self.size += 1
    
    def prepend(self, data):
        new_node = Node(data)
        new_node.next = self.head
        self.head = new_node
        self.size += 1
    
    def delete(self, data):
        if not self.head:
            return False
        
        if self.head.data == data:
            self.head = self.head.next
            self.size -= 1
            return True
        
        current = self.head
        while current.next:
            if current.next.data == data:
                current.next = current.next.next
                self.size -= 1
                return True
            current = current.next
        return False
```

### 2.4 집합(Set) ADT 구현

```python
class Set:
    def __init__(self):
        self.items = []
    
    def add(self, item):
        """집합에 요소 추가 (중복 불허)"""
        if item not in self.items:
            self.items.append(item)
    
    def remove(self, item):
        """집합에서 요소 제거"""
        if item in self.items:
            self.items.remove(item)
    
    def union(self, other):
        """합집합"""
        result = Set()
        result.items = self.items.copy()
        for item in other.items:
            result.add(item)
        return result
    
    def intersection(self, other):
        """교집합"""
        result = Set()
        for item in self.items:
            if item in other.items:
                result.add(item)
        return result
    
    def difference(self, other):
        """차집합"""
        result = Set()
        for item in self.items:
            if item not in other.items:
                result.add(item)
        return result
```
